#  Ludoria

An interactive educational app designed to prepare children for the KG stage by building essential skills such as letters, numbers, colors, shapes, and problem-solving.

Through fun games, puzzles, and short videos, Ludoria supports children’s academic, social, and emotional development while boosting confidence and school readiness.
